xampp的版本问题

公司一直都在用含有mysql的xampp，结果前几天我去官网看的时候发现，竟然全部都改成MariaDB了。之后再找版本就不容易了。于是记录一下。方便以后查找

支持xp。2003的版本是1.8.2以下。down的时候选择最新的就可以了。

最后支持mysql的版本是5.5.28/5.6.12他们两个的版本号都是根据php的版本来的。具体来说就是对应的是php5.5 以及php5.6两个版本。所以用哪个看你的php版本支持了。
一下摘抄自xampp的blog

v5.5.28 / v5.6.12

  • Updated Apache to 2.4.16
  • Updated PHP to 5.5.28 / 5.6.12
  • Updated MySQL to 5.6.26
  • Updated phpMyAdmin to 4.4.14
