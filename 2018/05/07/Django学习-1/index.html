Django学习-1

折腾了一圈，又开始回来学习Django了。
之所以回来，首先是因为其他的小的框架都太过简略，配置其他的东西比较麻烦。再一个，现在老了。越来越懒了。能有集成的东西用。就尽量快点能达到目的。

先说说遇到的坑吧。
第一个，关于python的虚拟机virtualenv 的坑。virtualenv 默认是copy pip的。但是现在pip的版本有个问题，10.0以上的版本，不支持main方法了。于是在pycharm加载的时候，就会提示报错。

但是我手动的先把已经安装好的python3的pip降级成了9.0.1之后，再调用virtualenv 生成新的虚拟机的情况下，pip的版本依然是10.0以上。这就比较尴尬了。

所以需要virtualenv 生成的虚拟机底下再把pip降级一遍。但是注意。如果你打成

pip install pip==9.0.1

是会报错的。因为此时不知道你想降级成哪个。正确的方式是指定python路径，再指定pip版本

python.exe -m pip install --upgrade pip==9.0.1

注意，我的这个命令是在虚拟机下面的Scripts文件夹下执行的，所以没写python.exe的路径。
